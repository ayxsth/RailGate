## Prerequisite

- Java 8 or higher
- IntelliJ IDEA / Apache NetBeans / Eclipse

## Running the project

- Clone the repository

  ``git clone https://github.com/ayxsth/RailGate.git``
- Open the IDE
- Go to the **Open Project** section
- Run the project
